# Objectif de coordination MZD via Claude Code établi

L'apprenant vise à utiliser Claude Code comme coordinateur actif de la MZD (Maison du Zéro Déchet),
en lisant l'état réel de l'organisation dans Notion, Drive et Slack, et en appliquant l'ingénierie
des skills Matt Pocock pour garantir que chaque skill injecte le contexte depuis les vrais outils
avant de demander à Claude de raisonner.

**Implications** : Les prochaines leçons doivent être orientées pratique (écriture de skills concrets),
en partant du pattern "fetch-first → reason → act" comme fil conducteur.
