# gaspardViel/skills

Skills Claude Code personnels — extensions MCP du framework Matt Pocock.

## Installation

```bash
claude skills add gaspardViel/skills
```

## Skills disponibles

| Skill | Usage |
|---|---|
| `mcp-reference-docs` | Créer et maintenir des fichiers de référence pour les IDs d'outils MCP |
| `mcp-parallel-fetch` | Écrire un fetch parallèle multi-sources dans un SKILL.md |
| `mcp-error-handling` | Gérer les erreurs et résultats vides lors d'un fetch MCP |

## Origine

Ces patterns étendent le framework [mattpocock/skills](https://github.com/mattpocock/skills)
spécifiquement pour les contextes où Claude Code utilise des outils MCP
(Notion, Slack, Google Drive, etc.).
